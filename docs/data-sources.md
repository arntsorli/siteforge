# Data Sources And Attribution

## Hoydedata DTM1

- Provider: Kartverket / Geonorge / Hoydedata
- Dataset: DTM1, national 1 m digital terrain model
- Feed: `https://nedlasting.geonorge.no/geonorge/ATOM/hoydedata/datasett/DTM1.atom`
- Format: GeoTIFF tiles
- Default CRS advertised by the feed: ETRS89 / UTM zone 33N, EPSG:25833
- License metadata in the feed: NLOD
- Attribution shown by SiteForge: `Kartverket / Geonorge / Hoydedata DTM1`

Every generated terrain must store the exact source tile URL and license URL used for that run.

## Accuracy Warning

SiteForge is for rough architecture planning only. DTM data is bare-earth terrain and does not represent current vegetation, buildings, fences, retaining walls, or temporary site conditions. Users must verify dimensions and elevations through proper survey workflows before engineering or construction use.
