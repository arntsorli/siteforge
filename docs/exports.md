# Export Plan

## MVP

- GLB terrain mesh.
- GLB terrain plus user-created planning objects.
- Project JSON beside each export.
- Attribution and rough-planning warning in sidecar metadata.

## Future

- OBJ for broad modeling compatibility.
- DXF for 2D footprint exchange.
- IFC for BIM-oriented workflows.
- Blender handoff with GLB plus project JSON.
- SketchUp workflow documentation using GLB/OBJ reference meshes.
